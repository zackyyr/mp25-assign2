# Clientify

A simple Android application built with Kotlin to help estimate project costs based on working hours and hourly rate.

## Overview

Clientify is a lightweight tool designed to assist freelancers, developers, or small teams in calculating project costs quickly.

Instead of manually calculating estimates, users can input the number of working hours and their hourly rate, then instantly get the total cost.

## Features

- Input working hours
- Input hourly rate
- Calculate total project cost
- Simple and easy-to-use interface
- Instant calculation using button action

## How It Works

1. Enter the total number of working hours
2. Enter the hourly rate (cost per hour)
3. Tap the "Calculate" button
4. The app will display the total estimated cost

Formula used:

Total Cost = Working Hours × Hourly Rate
